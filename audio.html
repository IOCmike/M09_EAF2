Apartat 2.3. Àudio a la web

1. Quins són els formats d'àudio més utilitzats i per què?

Els formats d'àudio més utilitzats a la web son el .mp3 i .ogg perquè son formats d'àudio amb pèrdues i això permet reduir el seu pes considerablement. El format mp3 tot i tenir patent es el més estès i compatible amb més navegadors que el format .ogg.

2. Cerqueu informació sobre almenys 2 reproductors d'àudio que ens permetin fer llistes de reproducció (playlist), amb l'ús d’HTML5 i JavaScript, i comenta breument les seves diferències.

Splayer: Un reproductor per HTML5 amb un disseny molt cuidat i amb moltes característiques, pots afegir caratules, inclou un buscador, pot retransmetre radio i admet diferents llistes de reproducció alhora. Es de pagament.

Green Audio Player: Es un reproductor minimalista amb el que pots fer playlists i customitzar els controls. Es gratuït.
